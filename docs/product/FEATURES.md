# Petory 当前功能

更新日期：2026-06-17。

## 客户端

- 使用邮箱 Magic Link 登录，不提供密码登录。
- 上传一张宠物照片后，默认生成同一主体的 6 种姿势：待机、开心、专注、睡觉、提醒、生气。
- 不提供风格迁移选择，生成结果应保持原图主体身份和标志性细节。
- **每个账号终身仅可成功创建 1 只自定义宠物**；创建成功后不可重新生成；删除本地宠物后也不可再次创建（服务端记录保留）。
- 支持桌面陪伴、聊天、专注计时、久坐提醒、成长记录和宠物管理（含示例宠物与云端恢复导入）。
- 默认示例宠物使用统一橘白猫的半写实假 3D 六姿势资源。

## 管理端

- 管理员使用受限邮箱 Magic Link 登录。
- 提供数据概览、用户、设备、生成记录、日志和系统配置。
- 生成任务展示服务端生成图；新版客户端本地生成任务会上传压缩预览。
- 会员、支付、兑换码功能当前下线，管理端不展示也不开放相关 API。

## 服务端

- 提供用户与管理员 Magic Link、JWT、设备、生成、聊天和额度能力。
- 所有用户使用同一套当前额度和 6 姿势生成规则。
- 图像生成默认使用火山引擎 **Doubao-Seedream-4.5**（`ARK_IMAGE_MODEL=doubao-seedream-4-5-251128`）。
- 数据库中保留部分旧版权益字段用于兼容历史数据，不代表当前产品功能。
